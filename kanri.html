5. 5島「関連サイト」リンクの管理（amami.html のポップアップ）

表の各島「関連サイト」ボタンで表示するリンクは、JSON で保持し、管理画面から編集します。

  • 編集画面（island-links-manage.php）はPHPではパスワードロックしません。公開ページからはリンクしておらず、運用MENUで URL を共有する前提です。ブロックが必要なときは下記の IP 制限または Basic 認証を使ってください。
  • 管理画面URL（本番例）：https://violetfoal2.sakura.ne.jp/hp-amami-pr-1/island-links-manage.php … 運用者は MENU 一覧（運用用）から開くか、上記 URL を直接入力してください。
  • より強く閉じたい場合：island-links-admin-config.php の allowed_admin_ips に管理者の固定 IP のみを列挙するか、island-links-manage.htaccess.example を参考に Basic 認証を追加してください。
  • 任意：island-links-admin-config.sample.php を island-links-admin-config.php にコピーし、必要なら allowed_admin_ips を書く（ファイルがなくても編集画面は動きます）。
  • 保存データ：data/island_related_links.json（Webサーバが書き込める権限が必要）
  • 公開API（閲覧用）：island-related-links-api.php … amami.html がここから取得し、失敗時はページ内のフォールバックを使います。
  • 編集内容：島ごとにポップアップの見出し（定型はプルダウンから選択、「その他（手入力）」で自由入力）、表示名・URLの行一覧。空行は保存時に無視されます。
  • アップロード時は island-links-manage.php・island-links-admin-guard.php・island-related-links-storage.php・island-related-links-api.php など、管理に関わるPHPと data/ を揃えること。
